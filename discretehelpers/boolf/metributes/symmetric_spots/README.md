# symmetry

This property describes the symmetry of the Euler diagram.<br>
It is a list of fullspots, that are symmetric to each other.


## _niliko_ with mirror symmetry

`niliko.symmetric_cells == [[0, 12], [1, 14], [5, 6], [9, 10]]`
